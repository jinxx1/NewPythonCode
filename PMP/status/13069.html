1、An organization’s lessons learned database indicates that past major project issues havebeen quality failures. The project manager decides to be more thorough when developing a qualitymanagement process.What should the project manager do to ensure quality?

一家组织的经验教训数据库表明过去的重大项目问题是质量失败。项目经理在制定质量管理过程时决定做得更彻底。为确保质量，项目经理应该怎么做？

A、 Develop a quality management plan based on the project’s scope and requirements. 根据项目的范围和需求制定质量管理计划。

B、 Incorporate all quality control processes in the project. 将所有质量控制过程纳入项目中。

C、 Include all lessons learned from past projects in the project checklist. 在项目核对单中包含来自过往项目的所有经验教训。

D、 Add a new risk to the risk register. 在风险登记册中添加一个新风险。
