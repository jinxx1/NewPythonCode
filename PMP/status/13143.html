75、During a monthly status report meeting, the sponsor asks the project manager questions about deliverables and budget restraints that were already answered and clarified in the weekly written report. The project manager believes that the sponsor is not reading the e-mail with the weekly report attached. What should the project manager do?

D、 Print out the weekly report and leave it on the sponsor’s desk. 将每周报告打印出来并放在发起人的桌子上。
