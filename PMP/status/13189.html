121、A project is being executed when a few changes are brought to the attention of the team.The changes are needed, but everyone has different opinions on what they think is right. What should the project manager do?

C、 Meet with team members to review the issues and request changes when needed 与团队成员来开会审查这个问题，并在需要时请求变更

D、 Apply all requested changes to the project and communicate with the team 对项目实施所有请求的变更，并与团队沟通
